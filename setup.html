Configuración inicial de la base de datos

1

Configurar Firebase

Primero necesitas configurar Firebase en la consola:

Pasos para configurar Firebase:

  1. Ve a Firebase Console
  2. Crea un nuevo proyecto (ej: "facturaflow")
  3. En la página principal del proyecto, haz clic en el ícono </> para agregar una app web
  4. Ingresa un nombre para tu app (ej: "FacturaFlow Web")
  5. No habilites Firebase Hosting ahora
  6. Haz clic en "Registrar app"
  7. Copia la configuración de Firebase

3

Configurar Firestore Database

Crea la base de datos Firestore y aplica las reglas de seguridad:

// En Firebase Console, ve a:
// Firestore Database → Crear base de datos
// Selecciona "Comenzar en modo de prueba"
// Elige una ubicación cercana
// Haz clic en "Habilitar"

Reglas de seguridad para Firestore:

// Copia estas reglas en Firestore → Reglas
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Reglas para usuarios
    match /users/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
    
    // Reglas para clientes
    match /clients/{clientId} {
      allow read, write: if request.auth != null &&
        (resource.data.userId == request.auth.uid ||
        request.auth.token.role == 'accountant');
    }
    
    // Reglas para facturas
    match /invoices/{invoiceId} {
      allow read, write: if request.auth != null &&
        resource.data.userId == request.auth.uid;
    }
    
    // Reglas para presupuestos
    match /budgets/{budgetId} {
      allow read, write: if request.auth != null &&
        resource.data.userId == request.auth.uid;
    }
    
    // Reglas para facturas comerciales
    match /commercialInvoices/{invoiceId} {
      allow read, write: if request.auth != null &&
        resource.data.userId == request.auth.uid;
    }
  }
}

Configurar Storage

Crea Storage y aplica las reglas de seguridad:

// En Firebase Console, ve a:
// Storage → Comenzar
// Sigue las instrucciones (acepta los términos)
// Elige una ubicación
// Haz clic en "Listo"

Reglas de seguridad para Storage:

// Copia estas reglas en Storage → Reglas
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /invoices/{userId}/{allPaths=**} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}
